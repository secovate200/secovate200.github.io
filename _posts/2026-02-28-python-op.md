- 파이썬에서 제공하는 **연산자**는 C언어와 유사하지만 몇가지 특징적인 차이가 있다.
- 연산자는 크게 **산술연산자**,**비교연산자**,**논리연산자**,**비트연산자**등으로 나눌수 있다.

## 산술연산자
- 숫자 연산을 처리하는 기본적인 연산자.
- C언어의 연산자와 유사하지만 파이썬에서는 **타입에 관계없이** 연산이 가능하다.
- 연산의 결과가 자동으로 **동적타입**에 맞게 반환된다. 

```python
x = 10
y = 3

print(x + y)  # 덧셈
print(x - y)  # 뺄셈
print(x * y)  # 곱셈
print(x / y)  # 나눗셈
print(x // y)  # 정수 나눗셈
print(x % y)  # 나머지
print(x ** y)  # 거듭제곱

```

## 비교연산자
- 두 값을 비교하여 참(True) 또는 거짓(False)을 반환하는 연산자 이다. 
- 파이썬의 비교연산자는 C언어와 매우 유사하다
```python
x = 10
y = 5

print(x == y)  # 같음 (False)
print(x != y)  # 같지 않음 (True)
print(x > y)  # 크기 비교 (True)
print(x < y)  # 크기 비교(False)
print(x >= y)  # 크거나 같음 (True)
print(x <= y)  # 작거나 같음(False)
```
- 파이썬에서 비교 연산자는 C언어와 동일 하지만 **동적타입**에 따라 문자열이나 다른 자료형의 비교도 지원한다. 

## 논리연산자 
- 파이썬의 **논리연산자**는 `and`,`or`,`not`으로 구성된다.

```python
x = True
y = False

print(x and y)  # and 연산
print(x or y)  # or 연산
print(not x)  # not 연산

``` 
## 비트 연산자 
- **비트연산자**는 비트 단위로 연산을 수행한다. 

```python
x = 10  # 이진수 1010
y = 4   # 이진수 0100
print(x & y)  # 비트 AND
print(x | y)  # 비트 OR
print(x ^ y)  # 비트 XOR
print(~x)  # 비트 NOT
print(x << 2)  # 왼쪽 시프트
print(x >> 2)  # 오른쪽 시프트
```
