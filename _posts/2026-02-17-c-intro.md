## C언어 
- c언어는 컴퓨터에 특정한 명령을 내리기 위한 프로그래밍 언어중 세계적으로 많이 쓰이는 언어중 하나이다.
- 아래의 코드는 컴퓨터가 `Hello World`라는 문자열을 출력하도록 하는 코드이다.
```C
#include<stdio.h>
int main(void){
    printf("Hello World");
    return 0;
}
```
## 코드 분석
### 1. include file
```c
#include<stdio.h>
```
`#include`는 뒤에 나오는 파일을 포함하겠다는 의미이다.
- `stdio.h` 파일을 포함하겠다는 의미이다.
### main function
```c
int main(){}
```
- `main` 함수를 선언하겠다는 의미이다.
- 프로그램이 실행되면 main함수 내부에 있는 코드가 실행된다. 
### 3.printf function
```c
printf("Hello World");
```
- `printf`뒤 괄호 내부에 있는 문자열을 출력해주게 된다.
- "Hello World" 라는 문자열을 함수에 전달한 인자라고 한다.
### return statument 
```c
return 0;
```
- 컴퓨터에 0을 반환하고 main함수를 종료하겠다는 의미이다.
