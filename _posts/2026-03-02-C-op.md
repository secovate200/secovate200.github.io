- c언어에서 연산자는 **값과 값 사이의 계산이나 비교**를 수행할때 사용하는 기호이다. 

### 연산자의 종류
- 산술연산자

```c
#include<stdio.h>
int main(void){
    int a=10; 
    int b=3;
    int sum = a + b;
    int diff = a - b;
    int prod = a * b;
    int quot = a / b;
    int mod = a % b;
    return 0;
}
```

- 복합대입 연산

```c
#include<stdio.h>
int main(void){
    int x=5;
    x+=3; // x=x+3;
    x *=2; // x=x*2;
    x -=4; // x=x-4;
    return 0;
}
```
- 비교연산자

```c
int a=10;
int b=5;
int c= (a>b);
int d =(a==b);
int e = (a <= 10);
```

- 논리연산자
    - `&&` 는 양쪽 조건이 모두 참이여야 결과가 참이다.
    - `||` 는 하나라도 참이면 전체가 참이다. 

```c
int age = 20;
int hasTicket = 1; // 1은 참

int allowEntry = (age >= 18) && (hasTicket == 1); 

```



- 증감 연산자
    - `x++`는 **후위 증가**이다.
        - 먼저 값을 사용하고 나중에 증가한다.
    - `++x`는 **전위 증가** 이다.
        - 먼저 증가하고 값을 사용한다. 

```c
int x = 10;
int y = x++;  // y = 10, x = 11
int z = ++x;  // x = 12, z = 12
```
