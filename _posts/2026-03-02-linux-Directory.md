## 절대경로 vs 상대경로

- 절대경로: 루트(`/`)부터 시작하는 전체경로

    ```shell
    cd /etc/network
    ```

    - `/`로 시작 : 절대경로
    - 루트부터 정확히 지정 


- 상대경로: 현재 디렉토리 기준으로 한 경로
    
    ```shell
    cd ../Documents
    ```
    - `..`는 상위 디렉토리 
    - 현재 위치 기준으로 이동 

## 파일종류
- 리눅스에서 `ls` 명령어를 통해 파일 리스트를 출력하면 아래와 같이 나오게 된다. 

```shell
drwxrwxrwx 1 secovate secovate  4096 Feb 11 02:20 .
drwxrwxrwx 1 secovate secovate  4096 Mar  2 00:07 ..
-rwxrwxrwx 1 secovate secovate   318 Oct 27 06:50 hello.c
```
-  리눅스에서 파일은 다음과 같은 유형으로 나뉜다.

    | 종류 | 기호 | 설명 |
    | :--- | :---: | :--- |
    | 일반 파일 | `-` | 텍스트, 바이너리 등 |
    | 디렉토리 | `d` | 폴더 |
    | 심볼릭 링크 | `l` | 다른 파일을 가리키는 링크 |
    | 문자 장치 파일 | `c` | 터미널, 시리얼 포트 등 |
    | 블록 장치 파일 | `b` | 하드디스크, USB 등 |

## 루트 디렉토리 
- 리눅스는 모든 파일과 디렉토리가 **루트(`/`)** 하위에 위치한다. 

```shell
ls /
```

### 주요 디렉토리 

| 디렉토리 | 설명 |
| :--- | :--- |
| `/bin` | 기본 명령어 (`ls`, `cp` 등) |
| `/boot` | 부트로더, 커널 관련 파일 |
| `/dev` | 장치 파일 |
| `/etc` | 설정 파일 |
| `/home` | 사용자 홈 디렉토리 |
| `/lib`, `/lib64` | 시스템 라이브러리 |
| `/media`, `/mnt` | 외부 장치 마운트 |
| `/opt` | 추가 소프트웨어 설치 |
| `/proc` | 커널 가상 파일시스템 |
| `/root` | 루트 사용자의 홈 |
| `/sbin` | 시스템 명령어 |
| `/tmp` | 임시 파일 저장 |
| `/usr` | 사용자용 프로그램 |
| `/var` | 로그, 큐 등 가변 데이터 |
