## 비트연산
- 피연산자를 2진수로 표현하여 비트 단위로 연산한것을 비트 연산자라고 한다. 

## 논리연산
- 참 또는 거짓 값으로 연산을 수행하고 결과로 참 또는 거짓을 반환한다.

    | 논리연산자 | 설명|
    | ----- | ----- |  
    |x `||` y|  둘중 하나라도 참이면 결과는 참이다. (OR연산자) |
    |x `&&` y | 둘다 참이여야 참이다.(AND 연산자) |
    | `!x` | 참이면 결과는 거짓, 거짓이면 참입니다(NOT연산자) |

### 비트연산자
- 비트단위로 논리연산을 수행하는 연산자 이다. 

    | 비트연산자 | 설명 |
    | ---------| ---- |
    | x `|` y | 두 비트중 하나라도 1이면 결과는 1이다. (OR연산자 )|
    | x `&` y | 두 비트 모두 1이면 결과는 1이다( AND연산자 )|
    | x `^` y | 두 비트가 같으면 0 다르면 1이다. (XOR연산자)|
    | ~x      | 비트가 0이면 1 1이면 결과는 0으로 모든비트를 반전시킨다(NOT)|

### 시프트 연산자
- 비트를 특정 값만큼 왼쪽 , 오른쪽으로 이동한다.
- n만큼 시프트 한 결과는 $n^2$ 으로 곱하거나 나눈값과 같다.

| 시프트 연산자 | 설명 |
| :--- | :--- |
| **`x << n`** | - 비트를 `n`만큼 **왼쪽으로 이동한다.**<br>- 오른쪽 빈 칸은 모두 **0으로 채운다.**<br>- **$== x \times 2^n$** |
| **`x >> n`**<br>(산술 시프트) | - 비트를 `n`만큼 **오른쪽으로 이동한다.**<br>- 왼쪽 빈 칸은 가장 왼쪽에 있던 비트(**MSB**)와 동일한 비트 값으로 **채운다.**<br>- (양수는 양수, 음수는 음수로 **부호가 유지된다.**)<br>- **$== x / 2^n$** |
| **`x >>> n`**<br>(논리 시프트) | - 비트를 `n`만큼 **오른쪽으로 이동한다.**<br>- 왼쪽 빈 칸은 모두 **0으로 채운다.**<br>- (음수는 **부호가 유지되지 않는다.**) |
